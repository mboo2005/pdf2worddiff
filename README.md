# pdf2worddiff
各pdf转换工具转换效果对比。


# 物料
随机抽取了万科某季度财报一份，文件大小4.2M，发现好多网站在线转换最大2m，裁剪其中的3页转换，一页是表格加合并后的单元格，一个是无边框的表格，加上两页图文排版，大家可以看转的效果。
